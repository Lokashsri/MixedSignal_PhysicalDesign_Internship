## Project Overview

This repository contains all tasks completed during the **VSD Mixed-Signal Physical Design Internship**.

The internship focuses on learning and implementing the complete RTL-to-GDSII physical design flow using open-source EDA tools.

The workflow includes:

- Environment setup
- OpenLane installation
- SKY130 PDK setup
- OpenROAD flow
- RTL synthesis
- Floorplanning
- Placement
- Clock Tree Synthesis (CTS)
- Routing
- Magic layout generation
- DRC verification
- Antenna checking
- Layout generation
- Documentation
